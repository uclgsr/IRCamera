# Baseline Profile for IRCamera
# Generated baseline profiles improve app performance on Android 12+
# by pre-compiling critical code paths ahead of time.
#
# To generate a production baseline profile:
# 1. Add androidx.benchmark:benchmark-baseline-profile-gradle-plugin
# 2. Create macrobenchmark tests that exercise critical user journeys
# 3. Run: ./gradlew :app:generateBaselineProfile
#
# Critical paths to profile:
# - App cold start (App.onCreate)
# - Camera initialization and first frame
# - Recording start/stop
# - Bluetooth device connection
# - UI navigation and Compose rendering
#
# For now, this is a placeholder. Actual profiles should be generated
# from real user journeys using Macrobenchmark.
#
# Example profile entries (these would be auto-generated):
# Lmpdc4gsr/core/App;->onCreate()V
# Lmpdc4gsr/core/RecordingService;->onStartCommand(Landroid/content/Intent;II)I
#
# See: https://developer.android.com/topic/performance/baselineprofiles
