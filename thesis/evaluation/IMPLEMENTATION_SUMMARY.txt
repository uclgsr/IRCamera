TEST IMPLEMENTATIONS
-------------------

1. GSR Synthetic Integrity Test
   - Generates 1280 samples (10s @ 128Hz) with known sine wave pattern
   - Validates: Data accuracy, timing, error metrics
   - Output: CSV with samples, JSON with analysis
   - Pass criteria: Error < 0.001 µS

2. GSR Real Sensor Continuity Test
   - Analyzes actual Shimmer3 GSR recording files
   - Validates: Sample rate, gaps, data completeness
   - Output: JSON with rate analysis and gap detection
   - Pass criteria: Rate deviation < 5 Hz, sample loss < 5%

3. Thermal Synthetic Capture Test
   - Simulates 50 frames (10s @ 5Hz) with timing validation
   - Validates: Frame rate, interval consistency
   - Output: CSV with frame timestamps, JSON with statistics
   - Pass criteria: FPS deviation < 0.5 Hz

4. Thermal Real Camera Test
   - Analyzes actual TC001 thermal camera recordings
   - Validates: Frame rate, drops, timing variance
   - Output: JSON with FPS analysis and drop detection
   - Pass criteria: FPS deviation < 5 Hz, drops < 5%

5. RGB Video Performance Test
   - Validates MP4 video and frame capture logs
   - Validates: File integrity, frame loss, timing
   - Output: JSON with video metadata and frame analysis
   - Pass criteria: Frame loss < 5%, file > 0.1 MB

6. File Integrity Validator
   - Validates CSV, video, and image files
   - Checks: File existence, size, format, completeness
   - Output: JSON with per-file validation results
   - Pass criteria: All files valid and complete

SCRIPTS
-------

run_all_tests.py
  - Executes all synthetic tests (no hardware required)
  - Executes real sensor tests (with data file paths)
  - Generates comprehensive JSON summary report
  - Returns exit code 0 (pass) or 1 (fail)

analyze_results.py
  - Scans output directory for test results

  - Outputs formatted text reports

USAGE EXAMPLES
-------------

# Run synthetic tests only
python3 scripts/run_all_tests.py

# Run complete suite with real data
python3 scripts/run_all_tests.py \
  --gsr-csv session/gsr_data.csv \
  --thermal-csv session/thermal_frames.csv \
  --rgb-video session/video.mp4 \
  --rgb-frames-csv session/rgb_frames.csv \
  --session-dir session/

# Generate analysis reports
python3 scripts/analyze_results.py

# Run individual test
python3 gsr_tests/gsr_synthetic_integrity_test.py

# Analyze specific file
python3 gsr_tests/gsr_real_sensor_continuity_test.py data.csv
